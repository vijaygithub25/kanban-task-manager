# 🚀 Kanban Task Management Web Application

A modern and interactive Kanban board application designed to manage tasks efficiently using a clean interface and drag-and-drop workflow.

---

## 🌐 Live Demo
https://vijaygithub25.github.io/kanban-task-manager/

---

## 🧠 Overview

This project implements a Kanban workflow where tasks are organized into three stages:

- To Do  
- In Progress  
- Done  

Users can create tasks, move them across stages, and track progress visually.

> Built to simulate real-world task management systems used in agile development environments.

---

## ✨ Features

- 📝 Create tasks with title and description  
- 🗑️ Delete tasks instantly  
- 🔄 Drag & drop tasks across columns  
- 📊 Real-time task count updates  
- 💾 Persistent storage using LocalStorage  
- 🎨 Clean and responsive dark-themed UI  

> Focused on simplicity, usability, and real-time interaction.

---
